### Baseline Cell Segmentation Pipeline 

## Project Goal 

The goal of this stage of the project is to establish a baseline for cell segmentation on the TRAgen
time lapse microscopy dataset. The baseline will later be compared against the first fram initialization and cell tracking approach 

The pipeline consists of there main stages:

1. Cell segmentation (pipeline.py)
2. Quantiative evaluation (evaluate.py) 
3. Visualization (visualize.py)

## 1. Segmentation - `pipeline.py`

### What it does 
Runs cellpose on all 83 frames automatically and saves predicted segmentation masks.

### Dataset 
- 83 grayscale microscopy frames generated by TRAgen (`img00000.tif`...`img00082.tif`)
- Ground-truth segmentation masks for each frame (`mask00000.tif`...`mask00082.tif`)  

### How it works 

**Loading frames:**
``` python3 image_files = sorted(glob.glob(os.path.join(DATA_DIR, "img?????.tif")))```

`glob` finds all matching TIFF files and `sorted()` ensures chronological order.


**Loading the first ground-truth mask:** 
```python3 first_mask = tifffile.imread(first_mask_path)```

The first mask is loaded for information purposes, it tells us how many cells 
are present at the start of the sequence:

``` python3 len(np.unique(first_mask)) - 1 # subtract 1 for background```

**Image normalizaton:** 
```python3 image = (image - image.min()) / (image.max() - image.min() + 1e-8)```

Each frame is normalzed to [0, 1] range. The small value `1e-8` prevents division by zero.

**Running Cellpose:** 
``` model = models.CellposeModel(gpu=True, pretrained_mode="cyto3") 
mask, flows, styles = model.eval(image, diameter=None, channels=[0, 0]```

`cyto3` is a pretrained Cellpose model for cytoplasm segmentation.
`diameter=None` means Cellpose estimates the cell size automatically
`channels=[0, 0]` means grayscale input

**Saving output:** 
Each prdicted mask is saved as 16-bit TIFF file: 

output_masks/pred_mask00000.tif 
output_masks/pred_mask00001.tif 
...
output_masks/pred_masks00082.tif

## Evaluation - `evaluation.py`

## What it does 
Compares predicted masks against TRAgen ground truth masks and computes accurracy metrics for each frame.

### Metrics computed 

** IoU (Intersection over Union):**
Measures the overlap between predicted and ground-truth cells regions.

IoU = intersection / Union 

A value of 1.0 means perfect overlap. Our paseline achieves **0.852**.

**Dice score:** 
Similar to IoU but weights overlap more heavily.

Dice = 2 * intersection (gt_pixels + pred_pixels) 

Our baseline achieves **0.920**.

** Precision:** 
Of all pixels predicted as cells, how many are actually cells?

Precision = intersection / pred_pixels

**Recall:** 
Of all actual cell pixels, how many did we correctly predict?

Recall = intersection / gt_pixels 

**F1 score:** 
Harmonic mean of precision and recal 

F1 = 2*precision*recal / (precision+recall) 

**Cell count error:** 
Absolute difference between number of cells in ground truth vs prediction.

### Output 
Results are saved to `results.csv` with one row per frame.

## 3. Visualization - `visualize.py`

### What it does 
Generates side by side comprasion images for selected frames showing:
- Original microscopy image 
- Ground-truth mask overlaid on the image 
- Predicted mask overlaid on the image 

### Frames visualized 
Frames 0, 10, 40, 60 and 82 are selected to show early, middle and late stages of the sequence.
